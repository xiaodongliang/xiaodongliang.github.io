Autodesk Forge 模型网页可视化技术导读

 

看完簡介及相關範例後，等不及打造自己的Forge應用？本页导引如何学习模型可视化技术。在開始前可以先依照Autodesk Forge基礎訓練工作坊 提供的步驟按部就班的完成入門基礎學習。

 

 

除了前面提到的基礎教學外，這邊有些額外讀物及資源供各位朋友參考。如前面所提到的，Autodesk Forge是一系列Web 服務所組成的強大的API服務平台，所採用的 API 風格為 RESTful，對此部份不熟悉的朋友可以點開『關於 REST API 的兩三事』的三個連結做一個初步的了解，若想快速測試 Forge API 可以透過 Postman 等REST API工具來協助操作（備註：撰寫Forge應用程式時用不到）。

 

如果是想在整合Forge Viewer的應用想多琢磨的朋友可以參考『JavaScript 入門資源』和『Three.js 入門資源』來學習相關技術。Forge Viewer與其他 Forge API 服務不一樣，他是一個 JavaScript 程式庫，用來在網頁上檢視模型，也可以透過 JavaScript 和 Three.js 來擴充 Viewer 的功能，亦可以將Viewer應用整合或鑲嵌到您的網站上。

 

此外，在學習過程中或是開發應用時遇到了問題，請不另隨時透過『Forge支援管道』提到的任一方式與我們的Forge 專家團聯繫，協助您解決 Forge 相關的技術問題！
